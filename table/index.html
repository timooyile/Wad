Web Application Development

Course Code: ITM 1131

Course Level: Year One, Semester One

Credit Unit: 4

Course Description

This course introduces students to Web technologies - HTML, CSS, JavaScript; CSS frameworks; JavaScript frameworks. Students will learn to implement modern web pages using HTML and CSS framework. The course also introduces students to full astack web application proframming using JavaScript.

Course Objectives

  1. Describe web technologies (HTML, CSS & JavaScript), CSS framewroks, and JavaScript frameworks.
  2. Implement a simple but modern web page with HTML and CSS.
  3. Build a simple but functional web application using JavaScript.

Learning Outcomes

  • Use web technologies (HTML, CSS & JavaScript), CSS frameworks, and JavaScript frameworks.
  • Design and creat a simple but functional web page using HTML and CSS framework.
  • Compose and build a simple full stack web application using JavaScript framework.
